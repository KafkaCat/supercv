# 数据备份与恢复说明

本应用所有数据存储在您的浏览器本地 IndexedDB 数据库中（数据库名为 `WonderCVDB`）。

## 自动备份
系统会在您点击“保存”按钮时自动创建版本快照。

## 手动备份（导出数据）
如果您需要迁移数据到另一台电脑或浏览器，可以使用以下方法（目前需通过开发者工具）：
1. 打开浏览器控制台 (F12)。
2. 选择 "Application" (应用) 标签页。
3. 在左侧 Storage -> IndexedDB 中找到 `WonderCVDB`。
4. 您可以使用第三方 IndexedDB 导出插件或手动复制数据。

*注意：后续版本将提供一键导出/导入 JSON 备份文件的功能。*

## 清除数据
如果需要重置应用，可以在浏览器设置中清除本站点的“Cookies 和网站数据”，或在控制台执行：
```js
indexedDB.deleteDatabase('WonderCVDB');
```
